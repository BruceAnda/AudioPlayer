This player is for Android 2.2+,and it's compiled under android API level-15.Now it is still on its middle stage and not that stable on all devices,but this developer will try to make it more compatible、functional and beautiful as well.

if you have tried ttpod music player,you will find this player is so similar with it.That's correct,this developer just imitates that appearance,but it's code may not similar with that as this developer;can not acess to ttpod resource.

this project starts on 2011-07-15 approximately,when this developer knew nothing about android and java.But with great interest and confidence,the developer started this discovery jounery .This project just indicates the developer's process and progress of learning.

As this project is not for commercial,but studying android application programming,so how the app looks like is not that important.

features:

  • similar to miui mediaplayer
  • HD album pictures and lyrics
  • support voice search
  • faster than most other players in the same level

Tips: while playing,downloading high definition album pictures may consumes much data and if you use GSM network,this will partly affect the performance;
